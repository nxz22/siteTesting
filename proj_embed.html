Overview

Developed a secure and innovative password system using the Circuit Playground Classic board. Utilized an ATmega32U4 processor and Arduino library. Employed an accelerometer-based authentication method.

Project Highlights

Transforming Security with Accelerometer-Based Authentication.

For our Embedded Systems Design class, our task was to utilize data from an accelerometer, built into the ATmega32U4 on a Circuit Playground Classic, and/or gyro to create a novel security system. The core idea was to record a hand movement sequence, which would serve as a unique key to unlock a resource. This approach marked a departure from traditional text-based passwords, venturing into the realm of motion-based authentication.

Hardware Configuration

The hardware configuration featured the Circuit Playground Classic Board, which employed the ATmega32U4 processor, crucial for our system's operation. An accelerometer was also integral, capturing motion data, which was the basis of our password system.

Software Architecture

Our software architecture utilized the Arduino Framework, specifically the Arduino.h and SPI.h libraries, providing a solid foundation for programming the ATmega32U4 processor. It featured a modular design, where code was structured into functions for better clarity and maintainability.

Data Handling and Processing

Data recording and filtering were implemented through functions like lowPassX(), lowPassY(), and lowPassZ() to filter accelerometer data. This data was recorded into Current_Pass_X/Y/Z and In_Progress_Pass_X/Y/Z arrays for the saved and attempted passwords, respectively.

Performance Optimization

For performance optimization, the use of pointers for memory efficiency was considered instead of direct array referencing.
